What is HTML?


HTML stands for HyperText MarkUp Language. It's a markup language that web developers use to structure and describe the content of a webpage (not a programing language).

HTML consists of elements that describe different types of content: paragraphs, links, headings, images, video etc. Web browsers understand HTML and render HTML code as webs.

In HTML each elements is made up of 3 parts.

1.Opening Tag
2.Closing Tag
3.The Actual Element
